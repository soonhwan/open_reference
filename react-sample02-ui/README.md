## BRANCH
- master: 기본
- storybook: 스토리북 포함 버전

### BRANCH 전략
- storybook은 독자적으로 운영하되, master의 바뀌는 내용들을 반영한다.
  - storybook을 checkout 하고, master 브랜치 우클릭 후 Merge into 'storybook'으로 머지
  - storybook의 바뀌는 내용들은 master에는 영향을 주면 안된다.
